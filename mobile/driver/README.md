# Smart Valet - Driver Mobile App

Flutter mobile application for valet drivers.

## Setup

1. Install Flutter SDK (>=3.0.0)
2. Install dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```

## Features

- Driver login (separate from web apps)
- Driver dashboard
- Location-based access
- Parking operations (to be implemented)

## Configuration

Update `lib/services/api_service.dart` with your backend API URL.
